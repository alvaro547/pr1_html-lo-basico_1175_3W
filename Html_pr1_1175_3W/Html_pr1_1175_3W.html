¿Qué es HTML?

HTML (Hypertext Markup Language) es el lenguaje estándar para crear páginas web. Es un lenguaje de marcado que estructura el contenido de la página mediante etiquetas.

Elementos básicos de HTML

  • Etiquetas: Elementos que se usan para marcar el contenido.
  • Atributos: Propiedades que se pueden agregar a las etiquetas para modificarlas.
  • Comentarios: Son notas dentro del código que no afectan a la visualización.
